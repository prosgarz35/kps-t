ТЕХНИЧЕСКИЕ ДАННЫЕ

 

ПРИМЕНЕНИЕ

Вилочная стрелка с ручным обслуживанием предназначена для изменения направления движения по монорельсовому пути в левую или правую стороны. Вилочная стрелка соединяется с монорельсовым путем по принципу фланцевого соединения. Подвешивается стрелочный перевод за натяжные пластины 11. Для изменения направления движения переводится стрелочный остряк 1. Перед тем как перевести стрелочный остряк, надо с помощью рукоятки 12 разъединить блокировку 3 и затем с помо­щью ручного толкателя перевести стрелочный остряк 1. При повороте стрелочногоо стряка в действие приводятся возвратные защелки 8, которые механически соединены с рычагом управления с помощью троса управления 7. Ручной толкатель представляет из себя металлическую трубку диаметром 24–30 мм и длиной 0,8–1,2 м.

ПРИМЕНЕНИЕ

Вилочная стрелка с ручным обслуживанием предназначена для изменения направления движения по монорельсовому пути в левую или правую стороны. Вилочная стрелка соединяется с монорельсовым путем по принципу фланцевого соединения. Подвешивается стрелочный перевод за натяжные проушины 10. Для изменения направления движения переводится стрелочный остряк 1. Перед тем как перевести стрелочный остряк, надо с помощью рукоятки 11разъединить блокировку 7 и затем с по­мощью ручного толкателя перевести стрелочный остряк 1. Для удобства перевода стрелочного остряка с почвы выработки установлены рукоятки перевода стрелочного остряка 13, которые через блоки, установленные на раме, переводят стрелочный остряк. Ручной толкатель представляет из себя металлическую трубку диаметром 24–30 мм и длиной 0,8–1,2 м.
